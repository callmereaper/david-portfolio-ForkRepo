## :hammer_and_wrench: Installation & Set Up
1. Install correct node version
   
   ```sh
   node --version
   v18.18.2
   ```
   
2. Install dependencies
   
   ```sh
   npm install
   ```
   
3. Start the local server
   
   ```sh
   npm run dev
   ```
